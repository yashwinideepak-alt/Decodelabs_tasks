DECODELABS Internship - Week 1

This week focused on learning blockchain fundamentals and implementing a simple blockchain using Python.

Project: Simple Blockchain Implementation

Description:
Built a basic blockchain model to understand how blocks are linked using cryptographic hashes and how immutability works.

Features:
- Created blocks with data, timestamp, and hash
- Linked blocks using previous hash
- Implemented SHA-256 hashing
- Basic mining concept using nonce

Technologies Used:
- Python

Key Learnings:
- Blockchain structure and working
- Hashing and immutability
- Proof-of-concept mining
- How blockchain ensures data integrity

Outcome:
Successfully built a basic blockchain model in Python and understood core blockchain concepts.
